# Security policy

## Supported versions

Until Skillet reaches 1.0, only the newest published version receives security
fixes. A fix may require upgrading rather than backporting.

## Reporting a vulnerability

Do not open a public issue. Use GitHub's private vulnerability-reporting form:

https://github.com/jnyross/Skill-Manager/security/advisories/new

Include the affected version, platform, reproduction steps, impact, and any
suggested mitigation. This is a small personal project with no guaranteed
response time, but reports will be acknowledged as capacity allows. Please
allow a reasonable opportunity to investigate before public disclosure.
