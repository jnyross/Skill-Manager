# Support

Skillet is a small, independently maintained project without a service-level
agreement or guaranteed response time.

- Use GitHub Issues for reproducible bugs and focused feature requests.
- Include `skillet --version`, operating system, CPU architecture, Node/npm
  versions, observed behavior, and minimal reproduction steps.
- Use GitHub private vulnerability reporting for security issues; never put a
  suspected vulnerability or secret in a public issue.

General Claude Code, Codex, Node, npm, or Go support is outside this project's
maintenance boundary unless the problem is caused by Skillet.
